# Skin2 files (binary-free source)

This repository keeps skin sources text-only.

- `theme.xml` is committed.
- Required bitmap files are generated by `scripts/generate_skin_bitmaps.py` during build.

## Build `.vlt`

From repository root:

```bash
bash vlc-skins/premium-material-dark-minimal/build-vlt.sh
```

Generated output:

- `vlc-skins/premium-material-dark-minimal/dist/premium-material-dark-minimal.vlt`

## Install

- Linux: `~/.local/share/vlc/skins2/`
- Windows: `%APPDATA%/vlc/skins2/`

Then select the skin in VLC settings and restart VLC.
